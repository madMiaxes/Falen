# Falen

Недавно я начал делать свою первую большую игру, текущий вариант название - **Falen**

Здесь я буду записывать:
- Идеи, которые приходят ночью,
- Ошибки, из-за которых всё ломалось,
- Моменты, когда вдруг «зажглось» — и мир стал живым.
- Нестандартные(Возможно) варианты решений тех или иных технических моментов в Unity


Основные записи
- [`DESIGN/`](DESIGN/) — механики, лор, описание систем  
- [`IDEAS/`](IDEAS/) — черновики, идеи
- [`PROGRESS/`](PROGRESS/) — прогресс разработки  
- [`ART/`](ART/) — скриншоты, палитры, концепты  
___
Разработка видется на Unity, Blender, VS 2022
